MyBatis String 无需类型处理器，直传直用

注意事项

  1. 时区问题：确保服务器和数据库时区一致
  2. 空值处理：前后端都要做好 null 值处理
  3. 格式统一：全项目统一使用 yyyy-MM-dd HH:mm:ss 格式
  4. 数据库自动更新：如果配置了 ON UPDATE CURRENT_TIMESTAMP，应用层不需要手动更新 update_time_dt

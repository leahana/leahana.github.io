## 🚀 快速开始（开发者）

```bash
git clone --recurse-submodules https://github.com/leahana/leahana.github.io.git
cd leahana.github.io && git checkout hexo

pnpm install
cd themes/kratos-rebirth && pnpm install && pnpm run build && cd ../..

pnpm dev  # http://localhost:4000
```

## 🛠️ 技术信息

### 项目结构
```
source/_posts/          # 博客文章
├── tech/              # 技术文章
├── design_patterns/   # 设计模式
├── gaming/            # 游戏相关
└── mindset/           # 思考随笔
themes/kratos-rebirth/ # 主题子模块
```

### 命令速查
```bash
pnpm dev      # 本地开发
pnpm build    # 生成静态站点
pnpm preview  # 预览（含草稿）
```

```

```

## Obsidian 简介（基于 5W1H 框架）

### 1. What（是什么？）

- **定位**：基于 Markdown 的本地优先笔记软件
- **核心功能**：
  - 双向链接（双向引用笔记）
  - 知识图谱（可视化笔记关联）
  - 插件生态（900+ 扩展功能）
- **典型用途**：知识管理、写作、构建第二大脑

### 2. Who（目标用户？）

- 🧠 **知识密集型工作者**：学者、作家、研究员
- 🛠️ **技术爱好者**：偏好数据本地化、支持 Markdown 和开源插件的用户
- 🧩 **个性化需求者**：需高度自定义笔记流程的人群

### 3. When（何时推出？）

- **发布时间**：2020 年
- **迭代节奏**：持续更新，已推出移动端、Canvas 白板等功能

### 4. Where（适用场景？）

- **设备支持**：
  - 桌面端：Windows/macOS/Linux
  - 移动端：iOS/Android
- **存储模式**：
  - 默认本地存储（可搭配 iCloud/Dropbox 同步）
  - 支持端到端加密的付费云服务（Obsidian Sync）

### 5. Why（核心优势？）

- ✅ **数据主权**：完全掌控本地文件（存储为 `.md` 格式）
- ✅ **网状思维**：通过链接实现非线性的知识连接
- ✅ **零成本启动**：基础功能免费，无强制订阅
- ✅ **隐私保护**：无广告、无数据上传分析

### 6. How（如何使用？）

1. **基础操作**：
   - 用 `[[ ]]` 语法创建笔记链接
   - 通过 `Ctrl/Cmd + O` 快速跳转笔记

2. **进阶功能**：
   - 插件市场安装工具（如日记模板、看板管理）
   - 用 CSS 自定义主题样式

3. **协作场景**：
   - 通过 Git 版本控制管理笔记
   - 使用付费版 Obsidian Sync 实时同步

## 同类工具对比

| 工具 | 优势 | 局限性 |
|------|------|--------|
| **Roam Research** | 实时协作/块级引用 | 纯云端/价格高（$15/月） |
| **Logseq** | 开源免费/大纲笔记 | 移动端体验差 |
| **Notion** | 数据库强大/团队协作 | 依赖网络/导出困难 |
| **RemNote** | 内置记忆卡片系统 | 界面复杂/学习成本高 |
| **TiddlyWiki** | 单文件便携/完全开源 | 多设备同步需技术配置 |

## 选择建议

### 选择 Obsidian 如果：

- 需要完全控制数据存储位置
- 习惯 Markdown 原生语法
- 愿意通过插件逐步搭建工作流

### 考虑替代品当：

- 需要开箱即用协作 → **Notion**
- 专注大纲笔记 → **Logseq**
- 研究学术文献 → **Roam Research**

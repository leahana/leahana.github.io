# MCP 模型上下文协议

## 1. MCP 简介
[模型上下文协议](https://modelcontextprotocol.org/docs/concepts/architecture)（MCP）是一种标准化协议，使 AI 模型能够以结构化方式与外部工具和资源交互。它支持多种传输机制，以便在不同环境中提供灵活性。

### 1.1 MCP 本质
- 标准化协议实现AI模型与外设的结构化交互
- 类似Type-C接口的function call统一标准
- mcp-全屋智能家电标准，function call-开关/遥控器

## 2. MCP 实践

### 2.1 Java MCP 实现
Java MCP 实现遵循三层架构：

![Java MCP 架构](https://docs.spring.io/spring-ai/reference/1.0/_images/mcp/mcp-stack.svg)

#### 2.1.1 架构层次
- **客户端/服务器层**：McpClient 处理客户端操作，而 McpServer 管理服务器端协议操作。两者都使用 McpSession 进行通信管理。
- **会话层（McpSession）**：通过 DefaultMcpSession 实现管理通信模式和状态。
- **传输层（McpTransport）**：处理 JSON-RPC 消息序列化和反序列化，并支持多种传输实现。

#### 2.1.2 MCP 客户端
![MCP客户端架构|650](https://docs.spring.io/spring-ai/reference/1.0/_images/mcp/java-mcp-server-architecture.jpg)

MCP 客户端是模型上下文协议（MCP）架构中的关键组件，负责建立和管理与 MCP 服务器的连接。它实现协议的客户端，处理：

- 协议版本协商，确保与服务器兼容
- 用于确定可用功能的功能协商
- 消息传输和 JSON-RPC 通信
- 工具发现和执行
- 资源访问和管理
- 提示系统交互

##### 可选功能
- 根管理
- 采样支持
- 同步和异步操作

##### 传输方式
- 基于 Stdio 的传输，用于基于进程的通信
- 基于 Java HttpClient 的 SSE 客户端传输
- 用于反应式 HTTP 流的 WebFlux SSE 客户端传输

#### 2.1.3 MCP 服务器
![MCP服务器架构|650](https://docs.spring.io/spring-ai/reference/1.0/_images/mcp/java-mcp-server-architecture.jpg)

MCP 服务器是模型上下文协议（MCP）架构中的基础组件，用于为客户提供工具、资源和功能。它实现了协议的服务器端，负责：

##### 服务器端协议操作实现
- 工具曝光和发现
- 使用基于 URI 的访问进行资源管理
- 及时的模板配置和处理
- 与客户的能力协商
- 结构化日志记录和通知

##### 并发管理
- 并发客户端连接管理
- 同步和异步 API 支持

##### 传输实现
- 基于 Stdio 的传输，用于基于进程的通信
- 基于 Servlet 的 SSE 服务器传输
- 用于反应式 HTTP 流的 WebFlux SSE 服务器传输
- 用于基于 servlet 的 HTTP 流的 WebMVC SSE 服务器传输

### 2.2 Spring AI MCP 集成
Spring AI 通过以下 Spring Boot Starters 提供 MCP 集成：

#### 2.2.1 客户端 Starters
[客户端 Starters](https://spring-doc.cadn.net.cn/spring-ai/1.0.0-M6/api_mcp_mcp-client-boot-starter-docs.html)

#### 2.3.3 代码实现
根据官网文档案例：

![模型设置](../../assets/images/mcp/mcp-example.png)

##### 创建 Service
```java
@Service
public class McpService {
    // 这里可以添加 MCP 相关的业务逻辑
    // MCP related business logic can be added here
    @Tool(name = "getWeather", description = "获取天气信息")
    public String getWeather(String city) {
        return "The weather in " + city + " is sunny.";
    }
}
```

##### 注册 ToolCallbackProvider Bean
```java
@Configuration
public class McpConfig {
    // 这里可以添加自定义的配置
    // Custom configurations can be added here
    @Bean
    public ToolCallbackProvider serverTools(McpService mcpService) {
        return MethodToolCallbackProvider.builder()
            .toolObjects(mcpService)
            .build();
    }
}
```

##### 配置 application.yml
```yaml
spring:
  application:
    name: java-mcp-utils
  ai:
    mcp:
      server:
        enabled: true
        name: ${spring.application.name}
        type: SYNC
        mode: stdio
```

#### 2.3.4 项目打包
```zsh
mvn clean package 
```
